✨ Inspired by modern messaging apps like WhatsApp

React Chat App is a modern and responsive chat interface built using React. 
It supports dynamic message rendering with proper alignment for sender and receiver, 
along with profile display logic similar to real-world messaging applications.

The project focuses on building a clean UI and handling core chat functionalities 
like conditional rendering, component structuring, and state management.

🚀 Features:
- Real-time styled chat UI
- Sender & receiver message alignment
- Dynamic profile image rendering
- Responsive design using Tailwind CSS
- Clean and reusable component structure

🛠️ Tech Stack:
- React.js
- JavaScript (ES6+)
- Tailwind CSS

📌 Future Improvements:
- Backend integration (Node.js + MongoDB)
- Real-time messaging with Socket.io
- Authentication system
- Message status (seen/delivered)
